# Todo Agent

CLI todo list assistant powered by OpenAI Agents.

## Setup

Create a `.env` file with your OpenAI API key:

```
OPENAI_API_KEY=sk-...
```

## Scripts

| Command | Description |
|---------|-------------|
| `npm run init` | First-time setup: installs deps and starts the app |
| `npm start` | Run the todo agent |
| `npm test` | Run tests |
